## 🗓 週次チェック(5 分)

毎週 月曜の朝などに、以下を確認:

### 1. GSC 「サイトマップ」
- [ ] ステータスが「成功しました」か(取得できませんでした → トラブル)
- [ ] 検出されたページ数が前週と比べて増えているか

### 2. GSC 「ページ」(インデックス作成)
- [ ] **登録済み** ページ数の推移(増加 = 正常)
- [ ] **未登録** ページの理由を確認(エラーがあれば対処)
- [ ] エラー(クロール拒否・noindex・サーバエラー等)の有無

### 3. GSC 「検索パフォーマンス」
- [ ] **合計クリック数**(週次で増えていれば正常)
- [ ] **合計表示回数**(増加すれば検索順位が上がっている)
- [ ] **新しく上位に来たキーワード** がないか

## 🗓 月次チェック(30 分)

毎月 1 日などに、以下を確認:

### 1. GA4 「レポート」概要
- [ ] **アクティブユーザー**(月次推移)
- [ ] **PV 数**(月次推移)
- [ ] **平均滞在時間**(2 分超なら良質)
- [ ] **流入経路**(Organic Search が増えているか)

### 2. GA4 「ページとスクリーン」
- [ ] **トップ 10 PV ページ** を確認
- [ ] 期待していたページが上位に来ているか
- [ ] 意外なページが伸びていないか(=次のコンテンツヒント)

### 3. GSC 「検索パフォーマンス」(詳細)
- [ ] **クエリ別** ─ どのキーワードでクリックされているか
- [ ] **ページ別** ─ どのページが評価されているか
- [ ] **CTR が低いページ**(表示は多いがクリックされない → タイトル・description 改善余地)
- [ ] **平均掲載順位**(下がっていれば要対策)

### 4. GSC 「コアウェブバイタル」
- [ ] LCP / FID / CLS がすべて「良好」か
- [ ] モバイルでの問題がないか

### 5. GSC 「リッチリザルト」
- [ ] **FAQ・パンくず・コース** が認識されているか
- [ ] エラーがあれば修正

## 📊 主要 KPI(目標値)

### 短期(3 ヶ月後)
- 月間 PV: 1,000 〜 5,000
- インデックス済みページ: 全ページの 80% 以上
- 主要 5 ページが「URL は Google に登録されています」状態

### 中期(6 ヶ月後)
- 月間 PV: 5,000 〜 20,000
- 検索流入が全体の 50% 以上
- 主要キーワードで 30 位以内

### 長期(1 年後)
- 月間 PV: 30,000 〜 100,000
- 主要キーワードで 10 位以内が 5 件以上
- 平均滞在時間 2 分以上

## 🛠 トラブルシューティング

### 「インデックス未登録」が増えた
1. GSC「ページ」→「未登録の理由」を確認
2. 主な理由と対処:
   - **クロール済み・未登録** → コンテンツの質を上げる、内部リンクを増やす
   - **検出 - 未登録** → サイトマップ確認、URL 検査で手動申請
   - **代替ページ(canonical タグあり)** → 想定どおりなら問題なし
   - **noindex タグ付き** → 意図的でなければ修正

### 検索流入が突然減った
1. **Google アップデート** ─ Search Engine Land や SISTRIX で確認
2. **手動アクション** ─ GSC「セキュリティと手動による対策」
3. **サイトの問題** ─ Vercel デプロイログ、ステータス確認
4. 慌てず 1〜2 週間様子見、改善されなければ対策

### サイトマップが「取得できませんでした」
1. ブラウザで `/sitemap.xml` を直接確認
2. Vercel デプロイ完了後 24 時間待つ
3. 一度削除して再送信
4. それでもダメなら静的 sitemap.xml に切り替え(本サイトは対応済み)
