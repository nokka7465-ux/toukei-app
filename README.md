# 統計ロードマップ(Toukei-App)

> AIエンジニアになるための **統計・数学・Python** を一気通貫で学べる無料サイト。

数学基礎 → 統計学 → 機械学習 → AI 検定 の **5 フェーズ** で迷わず最短距離を走るカリキュラム。統計検定 4 級〜1 級・G 検定・E 資格・DS 検定・QC 検定など主要検定を完全網羅し、教科書・図解・演習・Python/R コード・チートシート・学習プラン計算・模試・ブログまですべて 1 つのサイトで完結します。

🌐 **公開 URL**: https://toukei-app.com/
📦 **リポジトリ**: https://github.com/nokka7465-ux/toukei-app

完全無料・会員登録不要・ブラウザ完結・PWA 対応。

## 1. サイトの軸

> **AIエンジニアになるための統計・数学・Python を一気通貫で学べるロードマップ**

## 5. 独自の学習体験

### 5-1. 教科書の質感

- **明朝体(Noto Serif JP)** + 紙風背景で『教科書を読む』体験
- **TextbookBlock 型の 13 ブロック種別**(段落・数式・図解・コード・直感ボックス・実務ボックス etc.)
- **Python / R コード併記** タブ切替 + コピーボタン付きで scipy / statsmodels / scikit-learn / R を提示
- **100+ 箇所の直感解説 💡 と実務例 🛠**

### 5-2. 図解で学ぶ統計

- **38 種の SVG 図解**(後述、`/figures`)
- 静的画像ではなく theme-aware・印刷対応・ダークモード追従
- 各図解は教科書本文中にも埋め込まれ、`/figures` ページから該当節へ直リンク

### 5-3. 動かして学ぶ統計

[/explore](https://toukei-app.com/explore) のインタラクティブ図解 5 種(後述)。

### 5-4. 統計計算ツール

[/tools](https://toukei-app.com/tools) で 5 種(信頼区間 / p 値 / サンプルサイズ / 検出力 / A/B テスト)。

### 5-5. 横断検索 + Phase フィルター

[/search](https://toukei-app.com/search) で教科書 / 用語集 / 公式集 / ブログを横断。`もしかして候補` 機能 + ロードマップの **5 フェーズで絞り込み**。

### 5-6. 全文 noindex の進捗系ページ

`/review` `/bookmarks` などユーザー個別ページは noindex。

## 6. 学習機能(進捗 / バッジ / 模試 / 復習 等)

すべて **localStorage に保存**(サーバー送信なし)。

| 機能 | URL / 場所 | 概要 |
|---|---|---|
| 学習進捗トラッキング | ホームの `ProgressDashboard` | トラック別正答率 ・ 連続学習日数 ・ 最長記録 |
| 🔥 ストリーク | 同上 | 3/7/30 日達成でバッジ |
| 🏆 バッジ | ホームの `AchievementsPanel` | 15 種類のバッジ獲得 |
| 🎯 今日の 1 問 | ホームの `DailyProblem` | 日付ハッシュで全 14 トラックから自動抽出 |
| ✏️ 演習問題 | `/quiz`, `/quiz/[level]` | 1 問 1 答 + 一覧モード + 時間制限 |
| 🎮 ランダム練習 | `/practice` | 全範囲 / 弱点集中 / 級選択 + 難易度・カテゴリフィルタ |
| 🎓 模試 | `/mock`, `/mock/[level]` | 本番形式の時間制限 + 合否判定 + 履歴 |
| 🩹 苦手問題復習 | `/review` | 間違えた問題を全トラック横断で集約 |
| ⭐ ブックマーク | `/bookmarks` | 問題 / 公式 / 用語に星マーク |
| 📅 学習プラン計算 | `/plan` | 目標日 + 週時間 → 各フェーズの完了予定日 |
| 🧭 級診断 | `/diagnose` | 3 問で適切な級を提案 |
| 💾 進捗エクスポート/インポート | `ProgressBackup` | JSON で別端末に持ち越し可能 |

## 16. コンテンツ生成の哲学

### 16-1. 「なぜ」を必ず書く
形式的な定義だけでなく、**直感解説 💡** + **実務例 🛠** をセットで配置。検定対策本との明確な差別化点。

### 16-2. 図と数式と動きの 4 軸
同じ概念を:
- **教科書本文**(明朝体で読む)
- **SVG 図解**(`/figures` で見る)
- **インタラクティブ**(`/explore` で動かす)
- **コード**(Python/R で再現する)

の 4 つの角度から扱う。

### 16-3. 全文検索可能
教科書・用語集・公式集・ブログ・コードまで含めて、すべての block type が検索対象。**Phase 別フィルタ** で絞り込みも可能。タイポにも『もしかして候補』で対応。

### 16-4. 完全無料・完全静的
ユーザーログイン不要、データ送信なし、計算もブラウザ内完結。学習体験は印刷・オフライン保存も可能。

### 16-5. 学習者を孤立させない
3 つの始め方カード(教科書から / 演習から / 模試で力試し) + 5 フェーズロードマップ + 学習プラン計算 + 進捗トラッキング + バッジで、**「次に何をすればいいか」を常に提示**。
